# OfoxAI 开发文档

一个 API Key，接入 100+ 大模型。

完全兼容 OpenAI / Anthropic / Gemini 官方 SDK，零改动迁移。

## 三大协议，统一接入

- **OpenAI 兼容** — `https://api.ofox.ai/v1/chat/completions`
- **Anthropic 原生** — `https://api.ofox.ai/anthropic/v1/messages`
- **Gemini 原生** — `https://api.ofox.ai/gemini/v1beta/models/...`
